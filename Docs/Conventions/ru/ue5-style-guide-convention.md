# Конвенция UE5 Style Guide

**Name (slug):** `ue5-style-guide-convention`  
**Класс:** `UUE5StyleGuideConvention`

Корневой пресет: **PascalCase** и набор правил **имени и размещения** ассетов по [UE5 Style Guide v2](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file).

Каждое правило имеет **RuleId** и страницу в `Docs/Rules/` (и `Docs/Rules/ru/` при наличии перевода).

Проектные конвенции обычно **расширяют** этот пресет; унаследованные правила документируются в доке **базовой** конвенции.
