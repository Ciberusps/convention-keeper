# Конвенция Earendil

**Name (slug):** `earendil-convention`  
**Класс:** `UEarendilConvention`

Расширяет **UE5 Style Guide** и добавляет правила **структуры папок** Earendil и переопределения именования (в т.ч. **AnimSequence** с префиксом `AS_` вместо `A_`).

Полный список базовых правил — в [ue5-style-guide-convention](../ue5-style-guide-convention.md).

Собственные правила и override’ы — в английской версии `earendil-convention.md` (основной текст документации).
