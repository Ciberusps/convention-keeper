# asset-naming-sound-cue

**Sound Cue** assets must use the prefix **A_** and the suffix **_Cue**.

**Valid:** `A_Footstep_Cue`, `A_Explosion_01_Cue`  
**Invalid:** `A_Footstep`, `Footstep_Cue`, `SC_Footstep_Cue`

Applies to assets under `Content/{ProjectName}`.

## Rationale

The [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) uses A_ prefix and _Cue suffix for Sound Cue to distinguish them from raw Sound Wave (A_ only).
