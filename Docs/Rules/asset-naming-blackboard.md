# asset-naming-blackboard

**Blackboard** assets must use the prefix **BB_**.

**Valid:** `BB_Enemy`, `BB_Shared_Combat`, `BB_Boss_Spider`  
**Invalid:** `EnemyBlackboard`, `Blackboard_Shared`, `MyBB`

Applies to assets under `Content/{ProjectName}` (project-wide). Only Blackboard Data assets are checked.

## Rationale

The [UE5 Style Guide 1.2.3 Artificial Intelligence](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#123-artificial-intelligence) uses the BB_ prefix for blackboards so they sort and search consistently.
