# asset-naming-font

**Font** assets must use the prefix **Font_**.

**Valid:** `Font_Roboto`, `Font_UI_Main`, `Font_Character_Dialogue`  
**Invalid:** `Roboto`, `MainFont`, `F_Roboto`

Applies to assets under `Content/{ProjectName}`. Only Font assets are checked.

## Rationale

The [UE5 Style Guide 1.2.11 User Interface](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1211-user-interface) uses Font_ for font assets so they sort and are clearly identifiable in the content browser.
