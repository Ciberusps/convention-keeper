# asset-naming-sound-mix

**Sound Mix** assets must use the prefix **Mix_**.

**Valid:** `Mix_Gameplay`, `Mix_Cutscene`, `Mix_Menu`  
**Invalid:** `Gameplay`, `SoundMix_Gameplay`, `SM_Gameplay`

Applies to assets under `Content/{ProjectName}`.

## Rationale

The [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) uses Mix_ for Sound Mix so they are easily identifiable.
