# asset-naming-dialogue-voice

Ассеты **Dialogue Voice** должны иметь префикс **DV_**.

**Верно:** `DV_Player`, `DV_Narrator`, `DV_Character_Bob`  
**Неверно:** `Player`, `Voice_Player`, `DialogueVoice_Narrator`

Правило действует для ассетов в `Content/{ProjectName}`.

## Зачем

В [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) для Dialogue Voice задан префикс DV_; рядом используется DW_ для Dialogue Wave.
