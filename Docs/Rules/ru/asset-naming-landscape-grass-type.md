# asset-naming-landscape-grass-type

Ассеты **Landscape Grass Type** должны иметь префикс **LG_**.

**Верно:** `LG_Grass`, `LG_Flowers`, `LG_Weeds_01`  
**Неверно:** `Grass`, `LandscapeGrass_Grass`, `GrassType_Grass`

Правило действует для ассетов в `Content/{ProjectName}`. Проверяются только ассеты типа Landscape Grass Type.

## Зачем

В [UE5 Style Guide 1.2.7 Miscellaneous](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#127-miscellaneous) для Landscape Grass Type задан префикс LG_, чтобы такие ассеты было проще находить при работе с ландшафтной листвой.
