# asset-naming-sprite

Ассеты **Sprite** (Paper 2D) должны иметь префикс **SPR_**.

**Верно:** `SPR_Character`, `SPR_Prop_Tree_01`, `SPR_Enemy_Idle`  
**Неверно:** `Character`, `Sprite_Character`, `MySprite`

Правило действует для ассетов в `Content/{ProjectName}`. Требуется модуль Paper2D.

## Зачем

В [UE5 Style Guide 1.2.8 Paper 2D](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#128-paper-2d) для спрайтов задан префикс SPR_.
