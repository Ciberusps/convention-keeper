# asset-naming-anim-montage

Ассеты **Animation Montage** должны иметь префикс **AM_**.

**Верно:** `AM_Attack`, `AM_Character_Hit_01`, `AM_Reload`  
**Неверно:** `Attack_Montage`, `Montage_Reload`

Правило действует по всему проекту в `Content/{ProjectName}`. Проверяются только ассеты AnimMontage.

## Зачем

[UE5 Style Guide 1.2.2](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#122-animations) задаёт префикс **AM_** для анимационных монтировок для удобного поиска и идентификации.
