# asset-naming-media-sound-wave

Ассеты **Media Sound Wave** должны иметь префикс **MSW_**.

**Верно:** `MSW_Stream_01`, `MSW_Ambient_Stream`, `MSW_Music_Intro`  
**Неверно:** `Stream_01`, `Media_Stream`, `SoundWave_Stream`

Правило действует для ассетов в `Content/{ProjectName}`. Требуется модуль MediaAssets (движок).

## Зачем

В [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) для Media Sound Wave задан префикс MSW_, чтобы отличать от обычного Sound Wave (A_).
