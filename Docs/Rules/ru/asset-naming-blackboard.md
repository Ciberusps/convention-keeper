# asset-naming-blackboard

Ассеты **Blackboard** должны иметь префикс **BB_**.

**Верно:** `BB_Enemy`, `BB_Shared_Combat`, `BB_Boss_Spider`  
**Неверно:** `EnemyBlackboard`, `Blackboard_Shared`, `MyBB`

Правило действует для ассетов в `Content/{ProjectName}`. Проверяются только ассеты Blackboard Data.
