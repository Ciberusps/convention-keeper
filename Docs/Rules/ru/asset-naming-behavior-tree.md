# asset-naming-behavior-tree

Ассеты **Behavior Tree** должны иметь префикс **BT_**.

**Верно:** `BT_Enemy_Melee`, `BT_Turret_01`, `BT_Boss_Phase1`  
**Неверно:** `EnemyTree`, `BehaviorTree_Enemy`, `MyBT`

Правило действует для ассетов в `Content/{ProjectName}`. Проверяются только ассеты Behavior Tree.
