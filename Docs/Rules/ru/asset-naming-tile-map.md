# asset-naming-tile-map

Ассеты **Tile Map** (Paper 2D) должны иметь префикс **TM_**.

**Верно:** `TM_Level_01`, `TM_Environment_Grass`, `TM_Dungeon_Walls`  
**Неверно:** `Level_01`, `TileMap_Level`, `Map_01`

Правило действует для ассетов в `Content/{ProjectName}`. Требуется модуль Paper2D.

## Зачем

В [UE5 Style Guide 1.2.8 Paper 2D](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#128-paper-2d) для tile map задан префикс TM_.
