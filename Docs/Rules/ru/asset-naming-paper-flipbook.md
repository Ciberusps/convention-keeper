# asset-naming-paper-flipbook

Ассеты **Paper Flipbook** должны иметь префикс **PFB_**.

**Верно:** `PFB_Character_Run`, `PFB_Enemy_Explode_01`, `PFB_Prop_Door`  
**Неверно:** `Run`, `Flipbook_Run`, `MyFlipbook`

Правило действует для ассетов в `Content/{ProjectName}` (по всему проекту). Проверяются только ассеты типа PaperFlipbook.

## Зачем

В [UE5 Style Guide 1.2.8 Paper 2D](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#128-paper-2d) для paper flipbook задан префикс PFB_.
