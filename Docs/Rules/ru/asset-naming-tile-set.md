# asset-naming-tile-set

Ассеты **Tile Set** (Paper 2D) должны иметь префикс **TS_**.

**Верно:** `TS_Environment`, `TS_Dungeon_Tiles`, `TS_Grass_01`  
**Неверно:** `Environment`, `TileSet_Environment`, `Tiles_Dungeon`

Правило действует для ассетов в `Content/{ProjectName}`. Требуется модуль Paper2D.

## Зачем

В [UE5 Style Guide 1.2.8 Paper 2D](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#128-paper-2d) для tile set задан префикс TS_.
