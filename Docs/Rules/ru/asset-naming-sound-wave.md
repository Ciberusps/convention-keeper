# asset-naming-sound-wave

Ассеты **Sound Wave** должны иметь префикс **A_**.

**Верно:** `A_Footstep`, `A_Explosion_01`, `A_Ambient_Wind`  
**Неверно:** `Footstep`, `SFX_Explosion`, `Sound_Ambient`

Правило действует для ассетов в `Content/{ProjectName}`.

## Зачем

В [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) для Sound Wave задан префикс A_; они сортируются и отличаются от Sound Cue (A_ + _Cue).
