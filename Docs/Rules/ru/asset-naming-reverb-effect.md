# asset-naming-reverb-effect

Ассеты **Reverb Effect** должны иметь префикс **Reverb_**.

**Верно:** `Reverb_Cave`, `Reverb_Hall`, `Reverb_SmallRoom`  
**Неверно:** `Cave`, `RE_Cave`, `ReverbEffect_Hall`

Правило действует для ассетов в `Content/{ProjectName}`.

## Зачем

В [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) для Reverb Effect задан префикс Reverb_.
