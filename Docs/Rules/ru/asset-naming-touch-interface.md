# asset-naming-touch-interface

Ассеты **Touch Interface Setup** должны иметь префикс **TI_**.

**Верно:** `TI_Mobile`, `TI_Default`, `TI_Shooter_01`  
**Неверно:** `Mobile`, `TouchInterface_Mobile`, `Touch_Mobile`

Правило действует для ассетов в `Content/{ProjectName}`. Проверяются только ассеты типа Touch Interface Setup.

## Зачем

В [UE5 Style Guide 1.2.7 Miscellaneous](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#127-miscellaneous) для Touch Interface Setup задан префикс TI_, чтобы такие ассеты было проще находить при настройке мобильного или сенсорного ввода.
