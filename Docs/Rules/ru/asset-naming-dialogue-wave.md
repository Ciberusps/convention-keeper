# asset-naming-dialogue-wave

Ассеты **Dialogue Wave** должны иметь префикс **DW_**.

**Верно:** `DW_Line_01`, `DW_Greeting_Player`, `DW_Cutscene_Intro`  
**Неверно:** `Line_01`, `Dialogue_Line_01`, `Sound_Line_01`

Правило действует для ассетов в `Content/{ProjectName}`.

## Зачем

В [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) для Dialogue Wave задан префикс DW_; рядом используется DV_ для Dialogue Voice.
