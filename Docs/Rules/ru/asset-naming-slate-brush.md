# asset-naming-slate-brush

Ассеты **Slate Brush** должны иметь префикс **Brush_**.

**Верно:** `Brush_Button_Normal`, `Brush_Icon_Check`, `Brush_Background_01`  
**Неверно:** `ButtonBrush`, `SlateBrush_Icon`, `SB_Check`

Правило действует для ассетов в `Content/{ProjectName}`. Проверяются только Slate Brush.

## Зачем

В [UE5 Style Guide 1.2.11 User Interface](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1211-user-interface) для Slate brush задан префикс Brush_.
