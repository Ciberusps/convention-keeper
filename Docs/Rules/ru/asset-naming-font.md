# asset-naming-font

Ассеты **Font** (шрифты) должны иметь префикс **Font_**.

**Верно:** `Font_Roboto`, `Font_UI_Main`, `Font_Character_Dialogue`  
**Неверно:** `Roboto`, `MainFont`, `F_Roboto`

Правило действует для ассетов в `Content/{ProjectName}`. Проверяются только Font.

## Зачем

В [UE5 Style Guide 1.2.11 User Interface](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1211-user-interface) для шрифтов задан префикс Font_; так их удобно искать в контенте.
