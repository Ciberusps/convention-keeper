# asset-naming-sound-concurrency

Ассеты **Sound Concurrency** должны иметь суффикс **_SC**. В гайде рекомендуется называть их по имени Sound Class (префикс не обязателен).

**Верно:** `Weapons_SC`, `Ambient_SC`, `UI_SC`  
**Неверно:** `Weapons`, `SC_Weapons`, `SoundConcurrency_Weapons`

Правило действует для ассетов в `Content/{ProjectName}`.

## Зачем

В [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) для Sound Concurrency задан суффикс _SC; имена рекомендуется брать от Sound Class.
