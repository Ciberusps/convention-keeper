# asset-naming-env-query

Ассеты **Environment Query** должны иметь префикс **EQS_**.

**Верно:** `EQS_FindCover`, `EQS_NearestEnemy`, `EQS_PatrolPoints`  
**Неверно:** `FindCover`, `EnvQuery_FindCover`, `MyEQS`

Правило действует для ассетов в `Content/{ProjectName}`. Проверяются только ассеты Environment Query.
