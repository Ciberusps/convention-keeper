# asset-naming-sound-cue

Ассеты **Sound Cue** должны иметь префикс **A_** и суффикс **_Cue**.

**Верно:** `A_Footstep_Cue`, `A_Explosion_01_Cue`  
**Неверно:** `A_Footstep`, `Footstep_Cue`, `SC_Footstep_Cue`

Правило действует для ассетов в `Content/{ProjectName}`.

## Зачем

В [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) для Sound Cue заданы префикс A_ и суффикс _Cue, чтобы отличать их от сырого Sound Wave (только A_).
