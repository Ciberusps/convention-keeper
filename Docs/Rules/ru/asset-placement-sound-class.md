# asset-placement-sound-class

Ассеты **Sound Class** должны находиться в папке с именем **SoundClasses**. В гайде для имени ассета не заданы префикс и суффикс.

**Верный путь:** `Content/{ProjectName}/Audio/SoundClasses/MySoundClass`, `Content/{ProjectName}/SoundClasses/UI`  
**Неверный путь:** `Content/{ProjectName}/Audio/MySoundClass`, `Content/{ProjectName}/Classes/SoundClass`

Правило действует для ассетов в `Content/{ProjectName}`.

## Зачем

В [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) указано, что у Sound Class нет префикса/суффикса и они должны лежать в папке SoundClasses. Это правило проверяет только размещение в папке.
