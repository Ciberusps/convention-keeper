# asset-naming-sound-mix

Ассеты **Sound Mix** должны иметь префикс **Mix_**.

**Верно:** `Mix_Gameplay`, `Mix_Cutscene`, `Mix_Menu`  
**Неверно:** `Gameplay`, `SoundMix_Gameplay`, `SM_Gameplay`

Правило действует для ассетов в `Content/{ProjectName}`.

## Зачем

В [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) для Sound Mix задан префикс Mix_.
