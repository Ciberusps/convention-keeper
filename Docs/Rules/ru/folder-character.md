# folder-character

В каждой папке персонажа `Content/{ProjectName}/Characters/{CharacterName}/` должны быть: **AI**, **Projectiles**, **Animations**, **Gyms**, **Data**, **Abilities**, **Materials**.

## Зачем

Единая структура по персонажу держит анимации, AI, способности и арт в предсказуемых местах и упрощает инструменты и онбординг.
