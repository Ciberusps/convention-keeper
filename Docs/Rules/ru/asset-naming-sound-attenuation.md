# asset-naming-sound-attenuation

Ассеты **Sound Attenuation** должны иметь префикс **ATT_**.

**Верно:** `ATT_Indoor`, `ATT_Outdoor_Large`, `ATT_Weapon_01`  
**Неверно:** `Indoor`, `SA_Indoor`, `SoundAttenuation_Outdoor`

Правило действует для ассетов в `Content/{ProjectName}`.

## Зачем

В [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) для Sound Attenuation задан префикс ATT_ для быстрого поиска в Content Browser.
