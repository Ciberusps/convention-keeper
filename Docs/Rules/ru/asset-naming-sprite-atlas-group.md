# asset-naming-sprite-atlas-group

Ассеты **Sprite Atlas Group** (Paper 2D) должны иметь префикс **SPRG_**.

**Верно:** `SPRG_Characters`, `SPRG_Environment_Props`, `SPRG_UI_Icons`  
**Неверно:** `Characters`, `Atlas_Characters`, `SpriteAtlas_UI`

Правило действует для ассетов в `Content/{ProjectName}`. Требуется модуль Paper2D.

## Зачем

В [UE5 Style Guide 1.2.8 Paper 2D](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#128-paper-2d) для sprite atlas group задан префикс SPRG_.
