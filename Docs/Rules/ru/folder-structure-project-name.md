# folder-structure-project-name

В `Content/{ProjectName}/` обязательны папки: **Characters**, **Maps**, **Core**, **SFX**, **VFX**, **AI**.

## Зачем

Одинаковая структура под папкой проекта упрощает поиск геймплея, арта и кода. Карты, ядро и контент персонажей разделены по назначению.
