# folder-core-ai

Папка `Content/{ProjectName}/Core/AI/` должна существовать.

## Зачем

Общие AI-ассеты (деревья поведения, блэкборды, сервисы) хранятся в одном месте в Core проекта, отдельно от AI конкретных персонажей.
