# asset-naming-slate-widget-style

Ассеты **Slate Widget Style** должны иметь префикс **Style_**.

**Верно:** `Style_Button_Primary`, `Style_ScrollBar`, `Style_ComboBox_01`  
**Неверно:** `ButtonStyle`, `WidgetStyle_Primary`, `S_Button`

Правило действует для ассетов в `Content/{ProjectName}`. Проверяются только Slate Widget Style.

## Зачем

В [UE5 Style Guide 1.2.11 User Interface](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1211-user-interface) для Slate widget style задан префикс Style_.
