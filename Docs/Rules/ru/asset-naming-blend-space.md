# asset-naming-blend-space

Ассеты **Blend Space** (1D и 2D) должны иметь префикс **BS_**.

**Верно:** `BS_Locomotion`, `BS_Strafe_1D`, `BS_UpperBody_01`  
**Неверно:** `BlendSpace_Locomotion`, `Locomotion_BS`

Правило действует по всему проекту в `Content/{ProjectName}`. Проверяются только ассеты BlendSpace и BlendSpace1D.

## Зачем

[UE5 Style Guide 1.2.2](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#122-animations) задаёт префикс **BS_** для обоих типов Blend Space для единообразной сортировки и поиска.
