# asset-naming-anim-montage

**Animation Montage** assets must use the prefix **AM_**.

**Valid:** `AM_Attack`, `AM_Character_Hit_01`, `AM_Reload`  
**Invalid:** `Attack_Montage`, `Montage_Reload`

Applies project-wide under `Content/{ProjectName}`. Only `AnimMontage` assets are checked.

## Rationale

[UE5 Style Guide 1.2.2](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#122-animations) assigns the **AM_** prefix to Animation Montages so they are easy to filter and identify.
