# asset-naming-sprite

**Sprite** (Paper 2D) assets must use the prefix **SPR_**.

**Valid:** `SPR_Character`, `SPR_Prop_Tree_01`, `SPR_Enemy_Idle`  
**Invalid:** `Character`, `Sprite_Character`, `MySprite`

Applies to assets under `Content/{ProjectName}`. Requires the Paper2D module.

## Rationale

The [UE5 Style Guide 1.2.8 Paper 2D](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#128-paper-2d) uses SPR_ for sprites so they sort and are clearly identifiable.
