# asset-naming-reverb-effect

**Reverb Effect** assets must use the prefix **Reverb_**.

**Valid:** `Reverb_Cave`, `Reverb_Hall`, `Reverb_SmallRoom`  
**Invalid:** `Cave`, `RE_Cave`, `ReverbEffect_Hall`

Applies to assets under `Content/{ProjectName}`.

## Rationale

The [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) uses Reverb_ for Reverb Effect so they are easily identifiable.
