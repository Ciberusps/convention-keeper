# asset-naming-dialogue-wave

**Dialogue Wave** assets must use the prefix **DW_**.

**Valid:** `DW_Line_01`, `DW_Greeting_Player`, `DW_Cutscene_Intro`  
**Invalid:** `Line_01`, `Dialogue_Line_01`, `Sound_Line_01`

Applies to assets under `Content/{ProjectName}`.

## Rationale

The [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) uses DW_ for Dialogue Wave so they are easily identifiable alongside Dialogue Voice (DV_).
