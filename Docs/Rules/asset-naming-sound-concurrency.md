# asset-naming-sound-concurrency

**Sound Concurrency** assets must use the suffix **_SC**. The style guide recommends naming them after a Sound Class (no prefix required).

**Valid:** `Weapons_SC`, `Ambient_SC`, `UI_SC`  
**Invalid:** `Weapons`, `SC_Weapons`, `SoundConcurrency_Weapons`

Applies to assets under `Content/{ProjectName}`.

## Rationale

The [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) uses suffix _SC for Sound Concurrency, and they should be named after a Sound Class.
