# asset-naming-slate-brush

**Slate Brush** assets must use the prefix **Brush_**.

**Valid:** `Brush_Button_Normal`, `Brush_Icon_Check`, `Brush_Background_01`  
**Invalid:** `ButtonBrush`, `SlateBrush_Icon`, `SB_Check`

Applies to assets under `Content/{ProjectName}`. Only Slate Brush assets are checked.

## Rationale

The [UE5 Style Guide 1.2.11 User Interface](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1211-user-interface) uses Brush_ for Slate brush assets so they sort and are clearly identifiable.
