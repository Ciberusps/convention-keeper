# asset-naming-slate-widget-style

**Slate Widget Style** assets must use the prefix **Style_**.

**Valid:** `Style_Button_Primary`, `Style_ScrollBar`, `Style_ComboBox_01`  
**Invalid:** `ButtonStyle`, `WidgetStyle_Primary`, `S_Button`

Applies to assets under `Content/{ProjectName}`. Only Slate Widget Style assets are checked.

## Rationale

The [UE5 Style Guide 1.2.11 User Interface](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1211-user-interface) uses Style_ for Slate widget style assets so they sort and are clearly identifiable.
