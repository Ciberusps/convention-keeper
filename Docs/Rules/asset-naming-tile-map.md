# asset-naming-tile-map

**Tile Map** (Paper 2D) assets must use the prefix **TM_**.

**Valid:** `TM_Level_01`, `TM_Environment_Grass`, `TM_Dungeon_Walls`  
**Invalid:** `Level_01`, `TileMap_Level`, `Map_01`

Applies to assets under `Content/{ProjectName}`. Requires the Paper2D module.

## Rationale

The [UE5 Style Guide 1.2.8 Paper 2D](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#128-paper-2d) uses TM_ for tile maps so they sort and are clearly identifiable (distinct from Texture suffix _M for Mask/Metallic).
