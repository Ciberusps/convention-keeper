# asset-naming-tile-set

**Tile Set** (Paper 2D) assets must use the prefix **TS_**.

**Valid:** `TS_Environment`, `TS_Dungeon_Tiles`, `TS_Grass_01`  
**Invalid:** `Environment`, `TileSet_Environment`, `Tiles_Dungeon`

Applies to assets under `Content/{ProjectName}`. Requires the Paper2D module.

## Rationale

The [UE5 Style Guide 1.2.8 Paper 2D](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#128-paper-2d) uses TS_ for tile sets so they sort and are clearly identifiable.
