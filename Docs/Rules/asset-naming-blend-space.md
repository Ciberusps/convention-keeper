# asset-naming-blend-space

**Blend Space** (1D and 2D) assets must use the prefix **BS_**.

**Valid:** `BS_Locomotion`, `BS_Strafe_1D`, `BS_UpperBody_01`  
**Invalid:** `BlendSpace_Locomotion`, `Locomotion_BS`

Applies project-wide under `Content/{ProjectName}`. Only `BlendSpace` and `BlendSpace1D` assets are checked.

## Rationale

[UE5 Style Guide 1.2.2](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#122-animations) assigns **BS_** to both Blend Space and Blend Space 1D so they sort and search consistently.
