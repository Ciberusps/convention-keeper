# asset-naming-sprite-atlas-group

**Sprite Atlas Group** (Paper 2D) assets must use the prefix **SPRG_**.

**Valid:** `SPRG_Characters`, `SPRG_Environment_Props`, `SPRG_UI_Icons`  
**Invalid:** `Characters`, `Atlas_Characters`, `SpriteAtlas_UI`

Applies to assets under `Content/{ProjectName}`. Requires the Paper2D module.

## Rationale

The [UE5 Style Guide 1.2.8 Paper 2D](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#128-paper-2d) uses SPRG_ for sprite atlas groups so they sort and are clearly identifiable.
