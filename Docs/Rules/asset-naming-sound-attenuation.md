# asset-naming-sound-attenuation

**Sound Attenuation** assets must use the prefix **ATT_**.

**Valid:** `ATT_Indoor`, `ATT_Outdoor_Large`, `ATT_Weapon_01`  
**Invalid:** `Indoor`, `SA_Indoor`, `SoundAttenuation_Outdoor`

Applies to assets under `Content/{ProjectName}`.

## Rationale

The [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) uses ATT_ for Sound Attenuation so they are easily identifiable in the Content Browser.
