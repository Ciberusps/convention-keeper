# asset-naming-material-function

**Material Function** assets must use the prefix **MF_**.

**Valid:** `MF_Utility_Noise`, `MF_Blend_Overlay`, `MF_VertexColor_Blend`  
**Invalid:** `M_Noise`, `MaterialFunction_Blend`, `NoiseFunction`

Applies to assets under `Content/{ProjectName}`. Only Material Function assets are checked.

## Rationale

The [UE5 Style Guide 1.2.5 Materials](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#125-materials) uses MF_ for material functions so they sort and are distinguishable from materials (M_) and material instances (MI_).
