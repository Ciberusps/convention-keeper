# asset-naming-media-sound-wave

**Media Sound Wave** assets must use the prefix **MSW_**.

**Valid:** `MSW_Stream_01`, `MSW_Ambient_Stream`, `MSW_Music_Intro`  
**Invalid:** `Stream_01`, `Media_Stream`, `SoundWave_Stream`

Applies to assets under `Content/{ProjectName}`. Requires the MediaAssets module (engine).

## Rationale

The [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) uses MSW_ for Media Sound Wave to distinguish them from regular Sound Wave (A_).
