# asset-naming-sound-wave

**Sound Wave** assets must use the prefix **A_**.

**Valid:** `A_Footstep`, `A_Explosion_01`, `A_Ambient_Wind`  
**Invalid:** `Footstep`, `SFX_Explosion`, `Sound_Ambient`

Applies to assets under `Content/{ProjectName}`.

## Rationale

The [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) uses A_ for Sound Wave so they sort and are clearly identifiable alongside Sound Cue (A_ + _Cue).
