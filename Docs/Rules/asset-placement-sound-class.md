# asset-placement-sound-class

**Sound Class** assets must be located in a folder named **SoundClasses**. The style guide does not require a prefix or suffix for the asset name.

**Valid path:** `Content/{ProjectName}/Audio/SoundClasses/MySoundClass`, `Content/{ProjectName}/SoundClasses/UI`  
**Invalid path:** `Content/{ProjectName}/Audio/MySoundClass`, `Content/{ProjectName}/Classes/SoundClass`

Applies to assets under `Content/{ProjectName}`.

## Rationale

The [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) states that Sound Class has no prefix/suffix and should be put in a folder called SoundClasses. This rule enforces the folder placement only.
