# asset-naming-dialogue-voice

**Dialogue Voice** assets must use the prefix **DV_**.

**Valid:** `DV_Player`, `DV_Narrator`, `DV_Character_Bob`  
**Invalid:** `Player`, `Voice_Player`, `DialogueVoice_Narrator`

Applies to assets under `Content/{ProjectName}`.

## Rationale

The [UE5 Style Guide 1.2.10 Sounds](https://github.com/Allar/ue5-style-guide/tree/v2?tab=readme-ov-file#1210-sounds) uses DV_ for Dialogue Voice so they are easily identifiable alongside Dialogue Wave (DW_).
