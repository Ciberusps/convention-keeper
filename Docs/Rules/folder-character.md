# folder-character

Each character folder under `Content/{ProjectName}/Characters/{CharacterName}/` must have: **AI**, **Projectiles**, **Animations**, **Gyms**, **Data**, **Abilities**, **Materials**.

## Rationale

A uniform structure per character keeps animations, AI, abilities, and art in predictable places and simplifies tooling and onboarding.
