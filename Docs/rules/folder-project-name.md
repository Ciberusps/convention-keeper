# folder-project-name

Under `Content/{ProjectName}/` the following folders are **required**: **Characters**, **Maps**, **Core**, **SFX**, **VFX**, **AI**.

## Rationale

A consistent project layout under the project name folder makes it easy to find gameplay, art, and code-related content. Maps, core systems, and character content are separated by purpose.
